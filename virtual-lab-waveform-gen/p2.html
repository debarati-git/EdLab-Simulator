Sawtooth Wave Generator Using Op-Amp & RC Network

Analog Electronics · μA741 Op-Amp · Lab 05-B

📋 Problem Statement

Design a sawtooth wave generator using an RC charging network and a comparator reset circuit. The capacitor charges linearly through R from Vcc. A comparator monitors the voltage and resets the capacitor when it reaches Vp, creating a repeating sawtooth ramp. Design for the specifications below:

Output Frequency
500Hz
Peak Voltage V_p
8V
Supply Voltage V_cc
+15V
Capacitor C
100nF

📐 Governing Equations

T = R · C · V_p / V_cc
Period (linear ramp approximation, V_p << V_cc)
R = T · V_cc / (C · V_p)
Charging resistor — rearranged for design
💡

For a truly linear sawtooth, the RC time constant τ = RC must be much greater than the period T. Calculate τ and check the ratio τ/T to assess how linear the ramp will be.

🎯 Your Task

1. Calculate R using R = T · V_cc / (C · V_p) for f = 500 Hz, V_p = 8 V, V_cc = 15 V, C = 100 nF.
2. Compute the time constant τ = RC and verify τ >> T for linearity.
3. Confirm the output period T = 1/f = 2 ms.
4. Build and verify the circuit in the simulator.
