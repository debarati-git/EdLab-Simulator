Lab 05-A · Theory Reference
Triangular Wave Generator
How It Works

A triangular wave generator is built by cascading two op-amp stages. The first stage is a Schmitt trigger (comparator with hysteresis) — it switches its output between +V_sat and −V_sat based on the voltage at its non-inverting input. The second stage is an integrator — it integrates the square wave from the comparator, converting it into a linearly rising and falling triangular ramp.

Output Waveform
Circuit Block Diagram

Stage 1 — Schmitt Trigger: The comparator output switches between ±V_sat. With equal feedback resistors R₁ = R₂, the threshold voltage is ±V_sat/2.

Stage 2 — Integrator: The square wave is applied to the inverting integrator (resistor R, capacitor C). The constant input voltage causes the capacitor to charge/discharge linearly, producing the triangular ramp at the output. The integrator output feeds back to the Schmitt trigger's non-inverting input, closing the loop.

Key Equations
f = 1 / (4 · R · C)
Output frequency (R₁ = R₂)
V_peak = V_sat · (R₁/R₂)
Peak amplitude of triangular wave
V_sat ≈ ±11 V
μA741 at ±12 V supply
Design Steps
1
Choose C (e.g. 10 nF) and target frequency f.
2
Solve R = 1 / (4·f·C) for the integrator resistor.
3
Set R₁ = R₂ so V_peak = V_sat ≈ 11 V.
4
Verify V_peak < V_cc rails — no clipping should occur.
Lab 05-B · Theory Reference
Sawtooth Wave Generator
How It Works

A sawtooth generator uses an RC charging network to produce a linearly rising ramp voltage. A capacitor charges through resistor R from the supply V_cc. A comparator monitors the capacitor voltage and, when it reaches the reference threshold V_p, rapidly discharges the capacitor back to zero via a reset switch — creating the characteristic fast-fall, slow-rise sawtooth shape.

Output Waveform
Circuit Operation

Charging phase: The capacitor C charges exponentially through R from V_cc. When V_p << V_cc, the initial portion of the exponential is approximately linear, producing a good ramp. The larger the ratio τ/T (where τ = RC), the more linear the ramp.

Reset phase: When the capacitor voltage reaches V_p, the comparator output switches and triggers a discharge path, rapidly pulling the capacitor back to 0 V. The cycle then repeats.

Key Equations
T ≈ R·C·V_p / V_cc
Period (linear approximation)
R = T·V_cc / (C·V_p)
Design equation for R
τ = R·C >> T
Linearity condition
Design Steps
1
Choose C (e.g. 100 nF), supply V_cc, peak V_p, and target f.
2
Compute period T = 1/f, then solve R = T·V_cc / (C·V_p).
3
Calculate τ = RC and verify τ is several times larger than T for a linear ramp.
4
Ensure V_p < V_cc so the comparator can reliably detect the threshold.
