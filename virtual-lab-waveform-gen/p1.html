Triangular Wave Generator Using Op-Amp Integrator

Analog Electronics · μA741 Op-Amp · Lab 05-A

📋 Problem Statement

Design a triangular wave generator by cascading a Schmitt trigger comparator and an op-amp integrator using the μA741. The comparator output at ±Vsat is fed into the integrator to produce a linear triangular waveform. Design for the specifications below:

Output Frequency
1kHz
Capacitor C
10nF
R₁ = R₂ (Schmitt)
10kΩ
Power Supply Rails
±12V

📐 Governing Equations

f = 1 / (4 · R · C)
Triangular wave frequency (R₁ = R₂ case)
V_peak = V_sat · (R₁ / R₂)
Peak amplitude of triangular output
💡

Since R₁ = R₂, the threshold ratio is 1, so V_peak = V_sat ≈ 11 V for μA741 at ±12 V. This is safely within the ±12 V rails — no clipping occurs.

🎯 Your Task

1. Calculate R using R = 1 / (4 · f · C) for f = 1 kHz, C = 10 nF.
2. Predict V_peak of the triangular output (V_peak = V_sat since R₁ = R₂).
3. Determine whether the output will clip at the ±12 V supply rails.
4. Build and verify the circuit in the simulator.
