Theory
Inverting Amplifier
What it does

An inverting amplifier scales the input signal by a fixed gain and flips its polarity. A positive input produces a negative output. It uses an op-amp with two resistors — R_in sets the input path, R_f provides negative feedback from output back to the inverting (−) input.

Voltage Gain
A_v = − R_f / R_in Negative sign = phase inversion
V_out = − (R_f / R_in) × V_in Output voltage equation
Virtual Ground

Because the non-inverting (+) input is grounded and the op-amp has negative feedback, it drives its output until v⁻ = v⁺ = 0 V. The inverting node is held at 0 V without being directly connected to ground — this is called the virtual ground. This means the full input voltage appears across R_in, and KCL gives: i₁ = i_f = V_in / R_in.

Key Points
🔁
Phase
Output is 180° inverted relative to input
⚡
Rail Limit
V_out cannot exceed the supply rails (±V_CC)
🎛️
Gain control
Change R_f to change gain — R_in sets input impedance
📍
Virtual Ground
Inverting (−) terminal stays at ≈ 0 V due to feedback
