Design an Inverting Amplifier for a Specified Output

Analog Electronics · μA741 Op-Amp · P2

📋 Problem Statement

Design an inverting amplifier using a μA741 op-amp with a ±12 V dual supply. This time the required output voltage is given — you must first determine the gain, then find R_f:

Input Voltage V_in (peak)
0.30V
Required Output V_out (peak)
−2.4V
Input Resistor R_in
2.2kΩ
Power Supply Rails
±12V

📐 Governing Equations

|A_v| = |V_out / V_in|
Derive gain from given output and input
R_f = |A_v| × R_in
Select feedback resistor for required gain
V_out = −(R_f / R_in) × V_in
Verify your answer
💡

Work backwards: the output spec tells you the gain you need. Once you have the gain, R_f follows directly from R_f = |A_v| × R_in. Always verify V_out is within the ±12V supply rails.

🎯 Your Task

1. Calculate the required gain: |A_v| = |V_out / V_in|.
2. Calculate R_f = |A_v| × R_in.
3. Verify that V_out = −(R_f / R_in) × V_in matches the spec.
4. Enter your answers, then build and verify in the simulator.
