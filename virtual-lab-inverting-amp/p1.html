Design an Inverting Amplifier for a Specified Gain

Analog Electronics · μA741 Op-Amp · P1

📋 Problem Statement

Design an inverting amplifier using a μA741 op-amp with a ±12 V dual supply. The circuit must amplify and invert the input signal with the following specifications:

Input Voltage V_in (peak)
0.50V
Required Gain |A_v|
10×
Input Resistor R_in
1.0kΩ
Power Supply Rails
±12V

📐 Governing Equations

A_v = − R_f / R_in
Voltage gain of inverting amplifier
V_out = −(R_f / R_in) × V_in
Output voltage (linear operation)
💡

At the virtual ground node (op-amp inverting input), KCL gives i₁ = i_f = V_in / R_in. This current flows through R_f to set V_out.

🎯 Your Task

1. Calculate R_f using R_f = |A_v| × R_in.
2. Predict V_out = −(R_f / R_in) × V_in.
3. Enter your answers in the Pre-Lab Worksheet.
4. Build and verify the circuit in the simulator.
